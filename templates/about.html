What is this?

A website for the sole purpose of creating high quality gif files from YouTube. The current solutions online are something that work great, but do not offer the functionality I was looking for.

This is broken down into three main functions, each of which has its own way of creating gifs based on what you are looking for. Each one will be covered below.


OneVideoOneGif

What this allows you to do is create one gif given your desired start and stop time from the YouTube video you provide.


OneGifPerVid

This functionality allows you to create one gif per YouTube video where you provide multiple start and stop times. It takes start/stop times you provide and merges them into one gif. The number of YouTube links you provide will dictate the number of gif files you receive


ManyGifPerVid

This is like the antichrist to the OneGifPerVid function. Instead of merging the multiple start/stop times into one gif this will create one gif per start/end time you provide. In short, the number of gifs this returns is going to be equal to the start/stop times you enter regardless of the number of links.
